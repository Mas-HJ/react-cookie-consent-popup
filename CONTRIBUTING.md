# Contributing

Thanks for your interest in contributing to **react-cookie-consent-popup**!

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Mas-HJ/react-cookie-consent-popup.git
cd react-cookie-consent-popup

# Install dependencies
yarn install

# Run tests
yarn test

# Build
yarn build
```

## Development Workflow

1. Fork the repository and create a feature branch from `main`.
2. Make your changes.
3. Add or update tests for any new behavior.
4. Ensure all tests pass: `yarn test`
5. Ensure the build succeeds: `yarn build`
6. Submit a pull request.

## Code Style

- TypeScript strict mode is enabled.
- Formatting is handled by Prettier — run your editor's format-on-save or check with `npx prettier --check .`
- Linting is handled by ESLint — check with `npx eslint src/`

## Reporting Issues

Please open an issue at [GitHub Issues](https://github.com/Mas-HJ/react-cookie-consent-popup/issues) with a clear description and steps to reproduce.
